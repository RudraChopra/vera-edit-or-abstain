## Access Definition

registered or held-out leakage, certificate decision, selected candidate,
deployment violation, retention value, radius, margin, or aggregate comparison.
Process status, elapsed time, candidate counts, hashes, array supports and
lengths, paired-harm reconstruction, split identity, code revision, and an
audit's valid/missing/invalid counts are structural checks when no scientific
metric is displayed.
